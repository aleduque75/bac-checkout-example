## Installation

```bash
git clone https://github.com/leandrocfe/bac-checkout-example.git
```

Run the commands:

```bash
cd bac-checkout-example
cp .env.example .env
composer install
```

Laravel sail:

```bash
sail up -d
```

Run the commands:

```bash
sail npm i
sail npm run dev
```
